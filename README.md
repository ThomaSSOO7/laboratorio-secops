# laboratorio-secops
Laboratório de Secops

Descrição do Projeto

Este repositório foi criado no âmbito do meu estágio em SecOps (Security Operations) e reúne exercícios práticos e pequenos projetos desenvolvidos ao longo da formação.

O principal objetivo é reforçar e aplicar conhecimentos em diferentes áreas de tecnologia, incluindo controlo de versões, automação e simulação de ambientes cloud.

Ao longo do desenvolvimento, estou a consolidar competências em:

Git e GitHub: gestão de versões, criação de branches, pull requests e resolução de conflitos
Python: desenvolvimento de scripts para automação, análise de logs e monitorização de sistemas
LocalStack: simulação de serviços AWS em ambiente local (como S3, Lambda e IAM)
Projeto final integrador: criação de uma solução de monitorização e auditoria de segurança

Tecnologias Utilizadas
Linux (Rocky Linux 8) – ambiente base de trabalho
Git & GitHub – controlo de versões e colaboração
Python 3 – automação e scripts de análise
Docker / LocalStack – simulação de infraestrutura cloud
